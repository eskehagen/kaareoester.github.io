Første gangs opsætning

For at kunne publicere direkte til hjemmesiden skal du oprette et GitHub adgangstoken. Det gøres kun én gang — bagefter gemmes det i din browser.

  1. Gå til github.com/settings/tokens/new
  2. Skriv et navn, fx "Kaare hjemmeside"
  3. Under "Expiration" vælg "No expiration"
  4. Sæt hak ved "repo" (øverst i listen)
  5. Klik "Generate token" og kopier det lange token
💡 Tokenet starter med ghp_ og er ca. 40 tegn langt
✓ Forbundet til GitHub
